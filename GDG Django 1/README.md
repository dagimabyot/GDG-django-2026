## Tasks / Questions

1. Write a Django model for a `Post` with `title`, `content`, and `published_date` fields.  
2. Write a simple Function-Based View that returns `"Hello Blog"`.  
3. Write the URL pattern to connect the above view to `/home/`.  
4. Write a Class-Based View that handles a GET request to display `"Welcome to Django CBV"`.  
5. Write a Function-Based View that fetches all posts from the database and returns their titles in an `HttpResponse`.
